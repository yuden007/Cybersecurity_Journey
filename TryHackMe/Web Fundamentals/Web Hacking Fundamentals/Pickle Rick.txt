ffuf (Fuzz Faster U Fool) is a web fuzzer for discovering hidden files, directories, and parameters on web servers.

Basic Usage:
    ffuf -u <URL> -w <WORDLIST>
    Example: ffuf -u http://example.com/FUZZ -w /path/to/wordlist.txt

Options:
    -u <URL>: Target URL with FUZZ as the placeholder.
    -w <WORDLIST>: Wordlist for fuzzing.
    -mc <STATUS_CODES>: Match specific HTTP status codes (e.g., -mc 200,301).
    -o <OUTPUT_FILE>: Save results to a file.

Gobuster is a directory and file brute-forcing tool commonly used for web application testing.

Basic Usage:
    gobuster dir -u <URL> -w <WORDLIST>
    Example: gobuster dir -u http://example.com -w /path/to/wordlist.txt

Options:
    -u <URL>: Target URL.
    -w <WORDLIST>: Wordlist for brute-forcing.
    -x <EXTENSIONS>: File extensions to search for (e.g., -x php,html,txt).
    -o <OUTPUT_FILE>: Save results to a file.
    -t <THREADS>: Number of concurrent threads (default: 10).
    -s <STATUS_CODES>: Filter results by HTTP status codes (e.g., -s 200,204,301).
    -r: Follow redirects.
    -k: Skip SSL certificate verification.

Example:
    gobuster dir -u http://10.49.188.177 -w /usr/share/wordlists/dirb/common.txt -o gobuster_results.txt -t 20
